# Glossary — Trademarks and programme names

> SSOT for consistent naming in UI, API documentation, and user-facing copy.

## Planet Mark

- **Canonical name:** **Planet Mark** (two words, capital **P** and **M**).
- **Do not use:** “Planmark”, “PlanetMark”, “planet mark” (sentence case in titles only where grammar requires), or “PLANET MARK” except in all-caps legal contexts.
- **Technical identifiers:** The codebase may use `planet_mark`, `planet-mark`, or `planetmark` as slugs, route segments, or JSON keys — that is intentional for machines only; user-visible strings should say **Planet Mark**.

## Other assurance schemes (reference)

- **Achilles / UVDB** — use this form in UI; scheme id `achilles_uvdb` in APIs.
- **ISO** standards — use full designation when known (e.g. **ISO 9001:2015**).

## Introduction

The Quality Governance Platform (QGP) is an enterprise-grade Integrated Management System (IMS) that unifies:

- **Quality Management** (ISO 9001:2015)
- **Environmental Management** (ISO 14001:2015)
- **Occupational Health & Safety** (ISO 45001:2018)
- **Information Security Management** (ISO 27001:2022)
- **Planet Mark Carbon Management**
- **UVDB Achilles Supplier Qualification**

### Key Benefits

- ✅ Single source of truth for all governance data
- ✅ Mobile-first employee self-service portal
- ✅ AI-powered analysis and recommendations
- ✅ Automated compliance evidence collection
- ✅ Real-time dashboards and analytics
- ✅ Offline capability for field workers

## Employee Portal

The Employee Portal is designed for mobile workers to quickly report incidents and track their submissions.

### Home Screen

After login, you'll see:
- Your name and email
- Quick action buttons
- Recent submissions (if any)

### Submitting a Report

1. **Select Report Type**
   - Incident / Accident
   - Near Miss
   - Customer Complaint
   - RTA (Road Traffic Accident)
   - SOS Emergency

2. **Fill in Details**
   - Brief title (required)
   - Description of what happened
   - Location (auto-detected or manual)
   - Severity assessment
   - Photos (use camera button)

3. **Submit**
   - Review your information
   - Click "Submit Report"
   - Note your reference number (e.g., INC-2026-0001)

### Tracking Your Report

1. Go to "Track My Report"
2. Enter your reference number
3. View status timeline:
   - Reported → Under Investigation → Actions Assigned → Closed

### Emergency SOS

For immediate emergencies:
1. Tap the red SOS button
2. Your location is automatically captured
3. Emergency contacts are notified
4. Follow on-screen instructions

### Offline Mode

When offline:
- Reports are saved locally
- Automatic sync when connected
- You'll see a yellow "Offline" indicator
- View pending reports count

## Core Modules

### Incidents

**Creating an Incident**
1. Click "+ New Incident"
2. Fill in required fields
3. Assign severity and type
4. Add location and witnesses
5. Upload photos/evidence
6. Submit

**Managing Incidents**
- View in Kanban or list view
- Filter by status, severity, type
- Assign to investigators
- Link to actions/CAPA
- Close with resolution details

### Complaints

**Logging a Complaint**
1. Click "+ New Complaint"
2. Enter customer details
3. Describe the complaint
4. Set priority
5. Assign owner

**Resolution Workflow**
- Investigate root cause
- Create corrective actions
- Respond to customer
- Close with customer feedback

### RTAs (Road Traffic Accidents)

**Recording an RTA**
- Vehicle details
- Driver information
- Third-party details
- Witnesses
- Photos/diagrams
- Insurance information

### Risks

**Risk Register**
- View 5x5 risk matrix heat map
- Filter by category, owner, status
- Link controls to risks
- Track risk trends over time

**Adding a Risk**
1. Describe the risk
2. Assess likelihood (1-5)
3. Assess impact (1-5)
4. Assign owner
5. Define controls

### Actions/CAPA

**Creating Actions**
- Link to source (incident, audit, risk)
- Set due date
- Assign owner
- Define verification criteria

**Tracking**
- Dashboard shows overdue items
- Email reminders sent automatically
- Evidence upload for closure

## Troubleshooting

### Login Issues

**Can't login with Microsoft SSO?**
- Ensure you're using your @plantexpand.com email
- Clear browser cache
- Try incognito/private mode
- Contact IT if persists

**Forgot password?**
1. Click "Forgot Password"
2. Enter your email
3. Check inbox for reset link
4. Set new password

### App Not Loading?

1. Check internet connection
2. Clear browser cache
3. Try a different browser
4. Check system status page

### Push Notifications Not Working?

1. Check browser permissions
2. Enable notifications in device settings
3. Re-subscribe in app settings
4. Check "Do Not Disturb" mode

### Report Not Submitting?

1. Check all required fields
2. Reduce photo size if uploading
3. Check internet connection
4. Report saves offline for later sync
